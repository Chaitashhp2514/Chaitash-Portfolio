# Chaitash Patel — Portfolio

A production-ready computer-engineer portfolio (React + FastAPI + MongoDB) with midnight-violet theme.

**📖 See [DEPLOYMENT.md](./DEPLOYMENT.md) for the full free-hosting guide.**

## Quick start
```bash
# Backend
cd backend && pip install -r requirements.txt
uvicorn server:app --port 8001 --reload

# Frontend (new terminal)
cd frontend && yarn install && yarn start
```

Edit all content in `frontend/src/mock.js`.
